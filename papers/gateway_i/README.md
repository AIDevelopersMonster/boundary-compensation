Claim boundary:

The manuscript does not reconstruct physical baths, identify true hidden environments, equate Choi rank with physical bath dimension, or claim that `CERTIFICATE_RESET` is a physical event. It provides deterministic finite-dimensional certification diagnostics for reduced-channel data under declared thresholds and tolerances.
