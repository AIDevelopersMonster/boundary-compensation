# BC-Origin V software companion v0.1.3

Finite simplicial Z2 shadow-gauge ensemble lab for BC-Origin V.

## Purpose

The software demonstrates:

- holonomy-weighted adjacency operators A_epsilon;
- trace-cycle identity Tr(A^3)=6 sum_triangle Omega_triangle K_triangle;
- finite partition functions Z(gamma);
- deterministic inhomogeneous-bond triangle weights;
- Wilson-loop finite-graph diagnostics;
- shadow-admissibility statistics for D_epsilon = Delta + eta A_epsilon;
- optional MCMC for exploratory larger graphs.

## Command

python origin_v/software/generate_bc_origin_v_figures.py --out origin_v/figures

## Warning

The software is a finite graph/simplicial laboratory. It is not a physical lattice-QFT simulator, not a QCD confinement simulator, and not an empirical spin-glass model.
