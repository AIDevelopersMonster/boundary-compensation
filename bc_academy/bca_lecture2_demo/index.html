Demo 02: Hermitian Operators and Projectors

Интерактивное чтение самосопряжённого оператора как структурированной суммы eigenvalues и spectral projectors.

A = A† λj(A) ∈ ℝ A = Σj λjΠj Πj2 = Πj, Πj† = Πj

Цель модуля

Модуль показывает, почему самосопряжённость делает спектр вещественным, почему projectors устойчивее отдельных eigenvectors, и как trace суммирует спектральную информацию с учётом кратности.

Все вычисления выполняются прямо в браузере. Внешние библиотеки не используются.

4. Trace как spectral count

Диагональная модель с кратностью показывает, что trace projectors считает размерность spectral layer.

A = λΠλ + μΠμ, Tr A = λ Tr Πλ + μ Tr Πμ.
