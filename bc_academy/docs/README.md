# BC-Academy documents

This directory collects repository-facing documentation for **BC-Academy**, the educational and methodological layer of the Boundary Compensation programme.

BC-Academy is not a new physical hypothesis branch. It is a curriculum path for learning the finite-dimensional mathematical apparatus developed in the BC corpus.

## Module map

1. **BC-Academy I: Linear Spectral Entry** -- spectra, near-zero modes, gaps and spectral sensitivity.
2. **BC-Academy II: Schur/Feshbach Response Operators** -- scalar compensation gaps to induced finite-dimensional gap operators.
3. **BC-Academy III: Finite-Resolution Access and Sector Weights** -- spectral windows, projectors, sector weights and forward atlases.
4. **BC-Academy IV: Inverse Response and Non-Identifiability** -- response-equivalence classes and identifiability thresholds.
5. **BC-Academy V: Quotient Representatives and Certified Atlases** -- quotient representatives, walls, dictionary tubes, certification, reset, gluing and layer-aware metrics.

## Claim boundary

BC-Academy documentation is pedagogical and methodological. It does not claim derivation of the observed cosmological constant, reconstruction of a true hidden sector, particle-mass interpretation of induced gaps, detector theory, RG flow, continuum QFT dynamics, physical time evolution, or empirical prediction.
