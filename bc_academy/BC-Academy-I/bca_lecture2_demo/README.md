# BC-Academy I — Demo 02

**Hermitian Operators and Projectors**  
Автономный демонстрационный HTML-модуль к лекции-статье 2.

## Запуск

Откройте `index.html` в любом современном браузере.

Внешние библиотеки не используются. Интернет не нужен.

## Содержимое

1. Проверка эрмитовости матрицы 2×2.
2. Интерактивное спектральное разложение `A = λ₁Π₁ + λ₂Π₂`.
3. Проверка свойств projectors: `Π² = Π`, `Π† = Π`.
4. Trace как устойчивое суммирование спектральной информации.
5. Мини-контроль понимания.

## Методологическая граница

Модуль обучает конечномерной линейной алгебре: самосопряжённым операторам, вещественности спектра, spectral projectors и trace. Он не вводит физических claims и не реконструирует hidden sector.
