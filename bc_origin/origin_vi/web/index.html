Model statement

Z(gamma)=sum_epsilon exp(sum_triangle beta_triangle(gamma) Omega_triangle).
For each sampled epsilon: D_epsilon=Delta+eta A_epsilon. Projection failure occurs when lambda_min(D_epsilon) <= delta.

This is a finite-dimensional BC-Origin projection-failure laboratory. It does not simulate physical particle annihilation, quantum measurement collapse, empirical vacuum dynamics, or continuum QFT.
