# Описание статьи без формул

Статья BC-Origin VI продолжает линию BC-Origin I-V. В предыдущей статье была построена конечная калибровочная модель теней: скрытый граф с треугольными ячейками, реберными знаками и статистическим весом, который задает распределение допустимых конфигураций голономий.

Шестая статья отвечает на следующий вопрос: когда скрытая конфигурация перестает давать локальную наблюдаемую тень?

Главная идея состоит в том, что материя-подобная тень существует не просто потому, что скрытая структура существует. Она существует только если оператор считывания остается в области допустимости. Если спектральный зазор оператора пересекает заданный порог, локальный протокол проекции перестает возвращать наблюдаемый объект. Это называется spectral admissibility collapse или finite-resolution projection failure.

Важное исправление версии v0.1.2: фрустрация больше не задается искусственной вероятностью случайного минуса. Конфигурации знаков берутся из ансамбля, построенного в BC-Origin V. Управляющий параметр - gamma. Фрустрация измеряется как свойство конфигураций, полученных из этого ансамбля.

Статья соединяет три слоя:

1. калибровочный ансамбль из BC-Origin V;
2. спектральный оператор считывания тени;
3. вероятность отказа локальной проекции.

В результате появляется новая диагностическая величина: вероятность того, что при данном gamma скрытая конфигурация не возвращает локальную наблюдаемую тень. Это не утверждение о физической аннигиляции частиц и не доказательство онтологии вакуума. Это строгая конечномерная модель отказа readout-протокола.

Программная часть включает Python-модуль, генератор фигур и HTML-лабораторию. Она показывает, как изменение gamma меняет распределение фрустрации, спектральный минимум и вероятность projection failure.

GitHub-указание включено в рукопись, README и метаданные пакета.
