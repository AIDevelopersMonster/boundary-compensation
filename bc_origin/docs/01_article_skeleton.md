# Boundary Compensation Origin I: Oriented Winding Shadows, Structural Scale Selection, and Signed Spectral Localization

## Abstract draft

This note introduces the first toy model of the BC-Origin program. The model studies hidden residual generators carrying an index-like residual type and an oriented winding number. Observable elementary shadows are defined as structural projections of these generators. In the one-generator case, the winding number decomposes into a scale channel `|n|` and an orientation channel `sign(n)`, while a phase-closure equation selects a discrete family of observable scale ratios. In the two-generator case, the orientation product `sign(n1*n2)` enters a signed spectral operator and produces orientation-controlled localization, an admissibility horizon, and a coupling-induced shadow gap. The result is a structural-existence model for a non-fundamental observed layer: observable scale and orientation data are generated by compatibility equations and shadow operators rather than assigned as free cutoffs.

## 2. Hidden residual generators and observable shadows

Define:

```text
R_hid = Z e_q + Z e_n
h = (q, n)
```

where `q` is an index-like residual type and `n in Z` is an oriented winding number.

The winding number has two observable channels:

```text
scale channel: |n|
sign channel:  sign(n)
```

Define the shadow map:

```text
Pi(h) = (Q(h), N(h), Sigma(h), S(h))
```

with

```text
Q(h)     = q
N(h)     = |n|
Sigma(h) = sign(n)
S(h)     = ell_h / L
```

Important: additive composition lives at the winding level `n`, not directly at the sign level. That is,

```text
n_total = n1 + n2
sign_total = sign(n_total)
```

not `sign(n1)+sign(n2)`.

## 3. One-generator closure model

Spectral function:

```text
lambda_q(ell) = mu_|q| / ell
mu_|q| = alpha*f(|q|) > 0
```

Closure equation:

```text
lambda_q(ell)*L + theta(|q|) = 2*pi*|n|
```

Admissibility condition:

```text
2*pi*|n| - theta(|q|) > 0
```

Scale relation:

```text
ell_h/L = mu_|q|/(2*pi*|n| - theta(|q|))
```

### Proposition 1: Oriented winding shadow decomposition

For a nonzero winding number `n`, the observable shadow decomposes into a scale channel `|n|` and an orientation channel `sign(n)`.

### Proposition 2: Structural scale selection

For fixed structural data `(q, n)` satisfying the admissibility condition, the observable scale ratio is uniquely selected by the closure equation.

### Proposition 3: Discrete shadow scale family

For fixed `q`, the admissible scale ratios form a discrete family indexed by `|n|`.

## 4. Two-generator signed spectral model

For two generators:

```text
h_1 = (q_1, n_1)
h_2 = (q_2, n_2)
```

Define denominators:

```text
d_i = 2*pi*|n_i| - theta(|q_i|)
s = sign(n_1*n_2)
```

Signed common-shift matrix:

```text
D_signed = [[d_1 + gamma*s, kappa],
            [kappa,         d_2 + gamma*s]]
```

Eigen-denominators:

```text
lambda_pm = (d_1+d_2)/2 + gamma*s +/- sqrt(((d_1-d_2)/2)^2 + kappa^2)
```

Observable scales:

```text
S_pm = mu_eff / lambda_pm
```

for positive `lambda_pm`.

### Proposition 4: Orientation-controlled shadow localization

For `gamma > 0`, same-oriented winding products (`s=+1`) shift denominators upward and reduce observable scale ratios. Opposite-oriented products (`s=-1`) shift denominators downward and increase observable scale ratios.

### Proposition 5: Admissibility horizon

A branch is localized only if `lambda_pm > 0`. When `lambda_pm = 0`, the corresponding scale ratio diverges and the branch ceases to define a localized observable shadow.

### Proposition 6: Coupling-induced shadow gap

The branch separation is

```text
lambda_plus - lambda_minus = 2*sqrt(((d_1-d_2)/2)^2 + kappa^2)
```

so the off-diagonal coupling produces a reproducible branch gap. This gap separates the branches; it does not automatically protect every branch from horizon crossing.

## 5. Structural coupling problem and BC-Origin II handoff

In BC-Origin I, `gamma` and `kappa` may be used as explicit operator parameters. To avoid fitted-number contamination, BC-Origin II must replace them with structural values generated by kernels or overlap operators.

Candidate structural coupling:

```text
u_n(phi) = exp(i*n*phi)/sqrt(2*pi)
A = (1 - Delta_S1)^(-1)
kappa_ij = <u_ni, A u_nj> = 1/(1 + (n_i-n_j)^2)
```

This turns `kappa_ij` into a structural overlap value instead of a fitted constant.

## 7. Interpretation

The model demonstrates a possible mechanism by which observed elementary structures may be scale-selected and sign-sensitive shadows of hidden oriented winding generators. It is a structural-existence model and a computational scaffold for BC-Origin, not an identification of the physical hidden sector.

## 9. Conclusion

BC-Origin I establishes the first practical BC-Origin building block: oriented winding shadows, structural scale selection, signed spectral localization, admissibility horizon, and coupling-induced branch gap. BC-Origin II should derive coupling coefficients structurally and extend the model to multi-shadow spectral geometry.
