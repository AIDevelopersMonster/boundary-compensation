For full identification, every inequivalent pair must be covered. For the declared task Q, only pairs with different Q-values belong to the universe.
With binary responses, a pair is robustly separated when the response gap 1 exceeds 2δ. Exact completion can survive after the robust certificate fails.
The same threshold library requires all N−1 probes in a fixed package, but an adaptive binary-search policy has worst-case depth ⌈log₂N⌉.
The local scalar problem is a vector-matroid basis problem. Rank two removes tangent ambiguity, but it does not exclude remote aliases.
If p∘T=p, the probe has zero gain on every T-generated witness pair. Breaking one orbit is not yet a global completion certificate.
