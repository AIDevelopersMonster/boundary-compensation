# BC-Spec L1-P04-P Publication Synthesis

## Working title

**Curved-Volume Quantization, Ordering Residuals, and Visibility Scaling in SU(2)_k Quantum 6j Geometry**

## Scope

This publication project synthesizes **BC-Spec L1-P04-P/N7--N25** into one journal-style article and a separate reproducibility supplement. It is a publication workspace, not a new numbered technical module.

## Research object

The project follows one controlled chain:

1. reconstruct exact spherical tetrahedral volume from registered group-valued holonomy data;
2. prove that a single local trace-cubic cannot encode exact curved volume;
3. build the character/skein operator bridge and freeze ordering candidates;
4. replace proxy comparison by an operator-adapted residual-frame certificate;
5. validate visibility scaling prospectively across levels and external label families;
6. isolate absolute-ray-scale effects and identify the resolution/deformation confounding boundary.

## Publication architecture

- **Main article:** theorem chain, no-go results, prospective validation, claim firewall.
- **Reproducibility supplement:** module registry, frozen configurations, gate reports, datasets, figures, hashes, and rebuild instructions.
- **Heavy source archive:** stored in Google Drive; GitHub remains the lightweight public reproducibility/gateway layer.
