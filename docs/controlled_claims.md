# Controlled claims

This repository supports the following controlled claims.

1. The Choi matrix of a finite-dimensional reduced channel is a canonical positive response object.
2. A thresholded Choi support layer defines an effective finite-resolution Kraus-support diagnostic.
3. A deterministic operator-norm tolerance and a threshold margin define a sound finite-dimensional certification rule.
4. If the spectral margin to a declared threshold is not larger than the tolerance, a pathwise continuation cannot silently keep the previous certificate.

## Non-claims

This repository does not claim to reconstruct the true physical bath, infer a unique hidden Hamiltonian, prove a physical phase transition, derive quantum gravity, or replace standard open-system formalisms.
