\begin{proposition}[Validated implementation ceiling]
\label{prop:validationceiling}
On the frozen independent test atlas, the single truncated-Taylor implementation reproduces the previously independent first-, second- and third-matrix-derivative calculations within the thresholds in Table \ref{tab:residuals}.  The validated ceiling is matrix derivative order three and logarithmic-speed derivative order two.
\end{proposition}

\begin{proof}
This is a finite computer-assisted verification over the complete declared test list.  For every carrier, the program checks channel admissibility, orthogonality, nonzero anchor speed, the three analytic observables, two reference steps and the corresponding skew-symmetry identities.  The maximum residuals are those reported in Table \ref{tab:residuals}.  Source code and machine-readable certificates accompany the paper.
\end{proof}
